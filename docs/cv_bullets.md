# CV Bullets

- Built Smart Meter Analytics, an end-to-end utility data project using Python, PostgreSQL, FastAPI, SQL, and Power BI documentation to monitor consumption, meter health, anomalies, and data quality.
- Designed a PostgreSQL star schema with date, customer, region, and meter dimensions plus fact tables for readings, meter health, and data-quality events.
- Implemented a configurable synthetic data pipeline generating 12 months of daily smart-meter readings for 1,000 customers and 1,200 meters.
- Created SQL analytics views for executive KPIs, daily and monthly consumption, regional data quality, meter health, and anomaly reporting.
- Developed validation checks for duplicate readings, negative consumption, invalid IDs, future timestamps, missing values, anomaly consistency, and health score ranges.
- Built a FastAPI backend exposing KPI, meter, region, anomaly, and data-quality endpoints with automatic OpenAPI documentation.
- Prepared a Power BI-ready semantic model package with DAX measures, relationship guidance, report page layouts, metric definitions, and dashboard screenshot checklist.
- Wrote recruiter-facing and business-facing documentation explaining architecture, business value, metric definitions, and self-service dashboard usage.
