# Recruiter Project Summary

I built Smart Meter Analytics, an end-to-end utility data project that processes synthetic meter readings, validates data quality, stores data in PostgreSQL, exposes business KPIs through FastAPI, and provides a Power BI-ready semantic model and reporting package.

The project simulates a utility company receiving daily smart-meter readings from residential, commercial, and industrial customers. It includes realistic missing readings, duplicate readings, delayed readings, estimated readings, inactive meters, and abnormal consumption spikes.

This project demonstrates software engineering, Python data pipelines, PostgreSQL star schema design, SQL analytics views, data validation, API development, Power BI modeling, and business documentation.
